# Learning Games

A collection of interactive learning games for kids, hosted on GitHub Pages.

## Games

- **ABC Game** - Learn the alphabet
- **Add It Up!** - Practice addition
- **Dragon Vault: Number Quest** - Practice rounding numbers

## Development

Run a local server to preview:

```bash
python3 -m http.server 8000
```

Then open http://localhost:8000.

## Deployment

The site deploys automatically to GitHub Pages on every push to `main` via GitHub Actions.
